# `planet-wars/src/alexandria/`

[**Alexandria**](https://gitlab.common-lisp.net/alexandria/alexandria)：**Common Lisp 可移植扩展函数库**，MIT/X11 一类宽松许可证。Melis 将其 **vendor** 进本仓库以降低外部 ASDF 安装负担。

本目录 **`doc/`** 含上游撰写的参考文档草稿；权威 API 请以 [Alexandria 手册](http://alexandria.common-lisp.dev/)为准。

本项目 **不手写逐函数长文**：每个 `.lisp` 的 **`def*` 清单**已由脚本生成——见 [`../../docs/symbols/INDEX.md`](../../docs/symbols/INDEX.md) 中凡路径以 `src/alexandria/` 开头的条目。
